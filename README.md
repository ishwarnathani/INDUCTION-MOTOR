# Induction Motor Analysis Web Application

A responsive web-based engineering analysis tool for induction motor performance evaluation.

## Features
- Equivalent circuit parameter calculation
- No-Load Test analysis
- Blocked Rotor Test analysis
- DC Test analysis
- Torque–Speed characteristic generation
- Circle Diagram visualization
- Rotor resistance variation analysis
- PDF/report generation
- Interactive graphical outputs
- Cross-device compatibility

## Technologies Used
- React.js
- Vite
- JavaScript
- MATLAB (for validation)
- HTML/CSS
- Vercel

## Workflow
1. Enter No-Load Test data
2. Enter Blocked Rotor Test data
3. Enter DC Test data
4. Generate equivalent parameters
5. Visualize torque–speed characteristics
6. Analyze rotor resistance variation effects
7. Download generated reports
